Get started

Copy into any project. Everything is in .opencode/.

Per project

# Copy the full toolkit
cp -R .opencode/ /path/to/your-project/

# Or just the config
cp .opencode/config.json /path/to/your-project/.opencode/config.json
  • Agents, skills, commands, rules all included
  • Project-level config overrides global
  • No global setup required

Global install

# Make available in every project
cp -R .opencode/skills/* ~/.opencode/skills/
cp -R .opencode/commands/* ~/.opencode/commands/
cp .opencode/config.json ~/.opencode/opencode.json
  • All 12 agents available globally
  • Shared skills and commands across repos
  • macOS, Linux, and Windows supported

Usage

# Start OpenCode — IT Leader activates automatically
opencode

# Or call subagents directly
@frontend-nuxt Buat halaman login dengan Nuxt UI.
@backend Add endpoint POST /api/auth/login dengan DTO validation.
@android Buat login screen dengan Jetpack Compose + ViewModel.
@flutter Buat screen product list dengan Bloc pattern.
