Install

One Command. Full Team.

Run this in your project directory. Works on macOS, Linux, and Windows.

quick start

Install in any project

terminal 
$ npx opencode-agent-kit init
# Run inside your project directory
# macOS / Linux / Windows (WSL)
  ✓ Copying .opencode/ configuration
  ✓ Merging opencode.json
  ✓ Installing dependencies
  ✓ Setting up agentmemory (53 tools)
  ✦ 31 agents ready. Run opencode to start.

# Windows (PowerShell)
PS> npx opencode-agent-kit init
Copies .opencode/
Merges existing config
Installs agentmemory
Auto-start with opencode

Upgrade

$ npx opencode-agent-kit upgrade

Check and apply the latest version.

Doctor

$ npx opencode-agent-kit doctor

Diagnose setup and fix common issues.

Uninstall

$ npx opencode-agent-kit uninstall

Clean removal from your project.
