Get Started

Use in Any Project

Copy the .opencode/ folder into your project and instantly unlock 12 agents.

1

Clone or Copy

cp -R .opencode/ /path/to/your-project/

Or copy individual parts: config, agents, skills, commands, rules.

2

Set Your OpenAI API Key

export OPENAI_API_KEY="sk-..."
3

Run OpenCode

opencode

The IT Leader agent activates automatically. Just describe what you need.

4

Or Call Agents Directly

@frontend-nuxt Buat halaman login dengan Nuxt UI.
@backend Add endpoint POST /api/auth/login.
@android Buat login screen dengan Jetpack Compose.

Install Globally

Make all agents, skills, and commands available in every project.

mkdir -p ~/.opencode/skills ~/.opencode/commands ~/.opencode/rules ~/.opencode/contexts
cp -R .opencode/config.json ~/.opencode/opencode.json
cp -R .opencode/skills/* ~/.opencode/skills/
cp -R .opencode/commands/* ~/.opencode/commands/
